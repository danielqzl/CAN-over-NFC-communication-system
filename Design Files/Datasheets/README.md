# ECD401 Datasheet Collection

This folder collects the datasheets and other reference material necessary for understanding the functionality of all parts and protocols used in the project.

## NFC
Contains NFC protocol specification (NTAG/NFC Type-V, ISO18092/NFCIP, etc) and implementation information.

## NTP5332
Contains the datasheet and application notes for the NTP5332 NTAG 5 Link used for wireless sensing of the keyswitch.

## ST25R3916B
Contains datasheet for the ST25R3916B NFC transceiver/reader/writer, application notes, as well as the datasheets for the ST-provided RFAL abstraction layer/driver.

## STM32L476RG
Contains datasheets and reference manual for the STM32L476RG microcontroller
